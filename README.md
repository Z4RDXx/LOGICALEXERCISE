#  Cálculo de IMC em C

Este é um projeto simples desenvolvido para exercitar os fundamentos da linguagem C, como entrada e saída de dados, manipulação de variáveis do tipo float e estruturas condicionais encadeadas.
 Funcionalidades:

    Recebe o peso (kg) e a altura (m) do usuário via terminal.

    Calcula o Índice de Massa Corporal (IMC).

    Informa a classificação do peso com base nos padrões da OMS.

    Exibe os resultados formatados com duas casas decimais.

 Tecnologias Utilizadas:

    Linguagem C: Foco em lógica de programação e controle de fluxo.

    GCC: Compilador utilizado para gerar o executável.

 O que eu aprendi neste exercício:

    Diferença entre switch e if/else: Entendi que o switch é para valores exatos, enquanto o if/else é ideal para validar intervalos de números (maior, menor, entre).

    Ordem de Execução: Fixei que o processamento (cálculo) deve ocorrer obrigatoriamente após a entrada de dados.

    Formatação de Saída: Uso de especificadores como %.2f para uma exibição de dados mais limpa.

    Princípio DRY (Don't Repeat Yourself): Refatorei o código para centralizar a exibição dos resultados, evitando repetições desnecessárias de printf.
